### 2.选择排序

1. 思路综述：两层循环构成，外层代表已经排好的第几个数据,内层用来寻找要找到的天选之子，然后交换位置就好
2. 优化策略：在每一个大循环中间设置一个变量，表示是否发生交换，如果没有交换则直接返回，因为已经排好序了。举个例子，假设数据只有第一个要发生交换，全部循环下去就会很浪费时间，所以要在中间条件符合的情况下进行停止

```c++
vector<int> choice_sort(vector<int> &v){
	int len = v.size();
	for(int i=0;i<len;i++){
		bool t = false;
		for(int j=i+1;j<len;j++){
			if(a[i] > a[j]){
                swap(a[i],a[j]);
                t = true;
            }
            //
            if(t == false) break;
		}
	}
}
```

```c++
//这种每次标记最小值，只发生一次交换
vector<int> choice_sort(vector<int> &v){
	for(int i=0;i<v.size()-1;i++){
		int t_mid = i;
		for(int j=i+1;j<v.size();j++)
			if(v[j] < v[t_mid]) t_mid = j;
		if(t_mid != i)swap(v[i],v[t_mid]);
	}
} 
```



### 3.冒泡排序

1. 思路综述：同样两层循环，这个和插入排序略有不同，见字如面，就像冒泡一样，每次循环就是下沉的过程，检测出一个最重的数据放到最后一位，以此类推。
2. 优化策略：和插入一样

```c++
vector<int> bubble_sort(vector<int> v){
    int len = v.size();

    for(int i=0;i<len;i++){
        bool t = false;
        for(int j=0;j<len-i-1;j++){
            if(v[j] > v[j+1]){
                swap(v[j],v[j+1]);
                t = true;
            }
        }
        if(t == false) break;
    }
    return v;
}
```

```c++
vector<int> insert_sort(vector<int> &v){
    int len = v.size();
    for(int i=0;i<len;i++){
        int key = v[i];
        int pos = i - 1;
        //这里要格外注意，pos必须得有限制条件
        //这里的目的是为了给要插入的牌留个位置
        while(pos >=0 && key<v[pos]){
            v[pos+1] = v[pos];
            pos--;
        }
        //名正言顺归位
        v[pos] = key;
    }
    return v;
}
```

### 5.快排（分治）

1. 思路综述：设置一个基准点，以基准点为标准，把数组分为左右两部分，然后递归到两边
2. 优点：时间复杂度O（nlogn），比较优秀，并且是原地排序算法，只需要常数级的额外空间
3. 缺点：如果是已经排好的数据或者所有元素相同，时间复杂度退化到O(n*n)。
4. 代码

```c++
void quick_sort(vector<int> &v,int i,int j){
    if(i >= j) return ;
    int ref = v[(i+j)/2];
    int left = i,right = j;
    while(left <= right){
        while(v[left] < ref) left++;
        while(v[right] > ref) right--;
        //注意这里是指针比较，不是数字比较。
        if(left <= right){
            swap(v[left],v[right]);
            left++;
            right--;
        }
    }
    quick_sort(v,i,right);
    quick_sort(v,left,j);
}
```

### 6.归并排序

1. 思路综述：用分治法逐渐分割，把长长的数组分为众多个长度为一的区间，此时每个小区间肯定是有序的，然后逐渐合并小区间，合并的时候开辟临时的空间，让要合并的小区间有序的进去，然后把临时空间里面有序的数据放入小区间合成的大区间，类推就是一整个有序的数组

```c++
//预留空间因为后面要用
vector<int> t;
void merge_sort(vector<int> &v,int i,int j){
    if(i >= j) return ;
    int mid = (i + j) / 2;
    merge_sort(v,i,mid);
    merge_sort(v,mid+1,j);
    //这里就体现了容器的好用处了，可以清空和重新定义大小
    t.clear();
    t.resize(j - i + 1);
    int index = 0;
    int m = i,n = mid + 1;
    while(m <= mid && n <= j){
        if(v[m] < v[n]) t[index++] = v[m++];
        else t[index++] = v[n++];
    }
    //接下来主要是为了把剩下的没比较到的元素插入容器
    while(m <= mid) t[index++] = v[m++];
    while(n <= j) t[index++] = v[n++];
    //把容器倒腾到数组里面
    for(int k=0;k<index;k++){
        v[k+i] = t[k];
    }
}
```

## kadane（卡丹算法）

1. 算法自述：

   如果按照暴力求解，时间复杂度就是O(N^2），kadane算法是每次计算以这个数为末尾的区间最值，接下来要找下一个数的任何区间，只需要比较**自己**与**自己和前面最值的和**就行，这样避免了大量无意义的判断，时间复杂度是O(N),极其高效。

2. 代码

```c++
int kadane(const vector<int> &a){
    int len = a.size();
    int curr = a[0];
    int maxx = curr;
    //判断以a[i]为末尾的情况：
    for(int i=1;i<len;i++){
        //重新开始或者延续上面
        curr = max(a[i],curr+a[i]);
        maxx = max(curr,maxx);
    }
    return maxx;
}
```

```c++
const int N = 1e5 + 10;
vector<bool> a(N,true);
vector<int> prime;
void euler(int x){
    a[0] = a[1] = false;
    for(int i=2;i<=x;i++){
        if(a[i] == true) prime.push_back(i);
        for(auto p : prime){
            //超过数据范围就跳过
            if(p * i > x) break;
            a[p * i] = false;
            //精髓在这里，每次都少筛一下，就不会出现重复操作
            if(i % p == 0) break;
        }
    }
}
```

## 欧拉函数

1. 简介：欧拉函数是数论里面重要的函数，可以用公式求出互质的数的个数。
2. <img src="E:\typora\PLAN\知识点.assets\image-20250812102533253.png" style="zoom: 50%;" />
3. 步骤：先分解质因数，然后 N的互质个数 == N * （1-(1/p1)) * (1-(1/p2)) * ……
4. 代码实现

```c++
#include<bits/stdc++.h>
using namespace std;
#define int long long
#define endl "\n"

const int N = 1e5 + 10;
vector<bool> isprime(N,true);
vector<int> prime;
map<int,int> cnt;

void init_prime(){
	isprime[0] = isprime[1] = false;
	for(int i=2;i<N;i++){
		if(isprime[i])prime.push_back(i);
		for(auto p : prime){
			if(p * i >= N) break;
			isprime[i * p] = false;
			if(i % p == 0) break;
		} 
	}
}

signed main(){
	init_prime();
	int num; cin >> num;
	int t = num;
	for(int i=0;i<prime.size()&&prime[i]*prime[i]<=t;i++){
		while(t % prime[i]==0) cnt[prime[i]]++,t /= prime[i];
	}
	if(t > 1)cnt[t]++;
    //次数上面的全都是学过的，下面小块才是心内容
	int ans = num; 
	for(auto i : cnt){
		ans = ans * (1 - (1.0 / i.first));
	}
	cout << ans << endl;
} 
```
